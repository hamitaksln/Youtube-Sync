### [you-sync.herokuapp.com](https://you-sync.herokuapp.com/)
Youtube-Sync is an app that provides you to watch Youtube videos synchronous and chat with your friends.
## Build and Run Client and Server

 - `npm run build-run`
 
 ## Client
 ### Build
 - `cd client`
 - `npm run install`
 - `npm run build`

 ### Run
 - `npm run dev`
 
 ## Server
### Run in development
 - `npm run server`

### Run in production
- `npm run start` 

## Technologies
### Frontend:
- React JS
- Next.js
- Redux
- Tailwind CSS

### Backend:
- Express
- Socket IO
- Axios
